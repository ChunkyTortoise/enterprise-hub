## ðŸ“ Step-by-Step Recording Process

### Preparation (5 minutes)
1. **Clean your browser**:
   - Hide bookmarks bar (Ctrl/Cmd + Shift + B)
   - Close unnecessary tabs
   - Use Incognito/Private mode (no extensions)
   - Set zoom to 100%

2. **Open the demo**:
   - Navigate to: https://enterprise-app-mwrxqf7cccewnomrbhjttf.streamlit.app/
   - Go to **ðŸ’° Margin Hunter** page
   - Clear any existing values (refresh page)

3. **Set up recording area**:
   - Window size: 1920x1080 or 1280x720
   - Remove browser address bar (F11 fullscreen)
   - Ensure good lighting (if using webcam overlay)

### Recording (10 minutes)
1. **Start recording** in your chosen tool
2. **Follow the storyboard** (60-sec or 15-sec version)
3. **Move slowly and deliberately**:
   - Type at normal speed (not too fast)
   - Pause 1-2 seconds after each action
   - Let charts fully load before scrolling
4. **Record 2-3 takes** (you'll pick the best one)

### Post-Production (15 minutes)
1. **Trim video**:
   - Cut unnecessary beginning/end
   - Target: 30-60 seconds final length
   - Remove any pauses or mistakes

2. **Add text overlays** (optional):
   - Use video editor (OpenShot, DaVinci Resolve, iMovie)
   - Add key messages from storyboard
   - Keep text on screen for 3-5 seconds

3. **Export**:
   - **MP4**: H.264 codec, 1080p, 30 fps, ~5-10 MB
   - **GIF**: 720p, 15 fps, optimized, <5 MB

4. **Optimize file size**:
   - **MP4**: Use HandBrake or FFmpeg to compress
   - **GIF**: Use Gifsicle or ezgif.com to reduce size
