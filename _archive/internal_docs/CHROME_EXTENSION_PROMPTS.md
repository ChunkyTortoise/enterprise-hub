## ðŸ’¡ TIPS FOR USING EXTENSION

### **Best Practices:**
1. **Open the reference file** in a split screen so extension can see it
2. **Go field-by-field** - Don't rush through multiple fields at once
3. **Verify character counts** before pasting
4. **Ask for reformatting** if bullets/formatting breaks
5. **Take breaks** - Do 1 platform at a time (Fiverr first = fastest)

### **What Extension CAN'T Do:**
- Upload images (you'll manually upload from `docs/screenshots/`)
- Select from dropdowns (you'll manually select skills)
- Record video (you'll use phone camera)
- Submit forms (you'll click "Publish" yourself)

### **What Extension CAN Do:**
- Read your template files
- Copy text to paste
- Verify character counts
- Reformat if needed
- Guide you step-by-step
- Catch errors before you submit
